# Симуляция работы майнеров (Miner Mining Simulation)

## Обзор

Этот проект моделирует систему добычи ресурсов майнерами.
В системе несколько майнеров добывают ресурсы в течение дня, а их заработок зависит от количества добытых ресурсов, сложности добычи, условий работы и эффективности.

Цель проекта — смоделировать доменную логику добычи ресурсов, распределения задач, расчёта заработка, бонусов и штрафов.

---

# Основная идея

В начале дня система содержит структуру данных, которая показывает, сколько ресурсов должен добыть каждый майнер.

Пример:

```go
Map<MinerId, ResourcesToMineToday>
```

Пример данных:

```go
{
  1: 50,
  2: 40,
  3: 70
}
```

Где:

* **ключ** — ID майнера
* **значение** — количество ресурсов, которое нужно добыть за день

---

# Расчёт заработка

Заработок майнера рассчитывается по формуле:

```bash
salary = resourcesAssigned × baseRate × difficultyCoefficient × equipmentCoefficient × experienceCoefficient
```

## Параметры

**resourcesAssigned**
Количество ресурсов, которое майнер должен добыть.

**baseRate**
Базовая ставка за единицу ресурса.

# Ограничения добычи

У каждого майнера может быть максимальный объём ресурсов, который он может добыть за день.

| Уровень | Максимум ресурсов |
| ------- | ----------------- |
| Junior  | 30                |
| Regular | 50                |
| Senior  | 80                |

Если майнеру назначено больше ресурсов:

* часть задач может перераспределяться
* либо переноситься на следующий день

---

# Бонусы

Если майнер добывает больше ресурсов, чем планировалось, он может получить бонус.

Формула:

```bash
bonus = extraResources × bonusRate
```

Пример:

```bash
bonusRate = baseRate × 0.7
```

---

# Штрафы

Если майнер не выполнил план добычи, может применяться штраф.

Пример формулы:

```bash
penalty = missingResources × baseRate × penaltyCoefficient
```

Альтернативный вариант:

```bash
penalty = sqrt(missingResources) × baseRate
```

---

# Типы ресурсов

Разные ресурсы могут иметь разную ценность.

| Ресурс  | Множитель |
| ------- | --------- |
| Coal    | 1.0       |
| Iron    | 1.3       |
| Gold    | 2.0       |
| Diamond | 3.5       |

Формула оплаты:

```bash
payment = baseRate × resourceMultiplier
```

---

# Поломки оборудования

Во время добычи может происходить поломка оборудования.

Возможные события:

```bash
NO_BREAK
MINOR_BREAK
MAJOR_BREAK
```

Если происходит поломка:

* снижается эффективность
* уменьшается итоговый заработок
* увеличивается время добычи

---

# Энергия майнера

Каждый майнер имеет запас энергии.

```bash
MinerEnergy
- maxEnergy
- energyPerResource
```

Если энергия заканчивается:

* добыча останавливается
* часть ресурсов остаётся недобытой

---

# Статистика майнера

Система может хранить статистику работы.

```bash
MinerStats
- resourcesMinedTotal
- failedAttempts
- rating
```

Рейтинг может влиять на коэффициенты заработка.

---

# Отчёт за день

После завершения рабочего дня формируется отчёт.

```bash
MinerDayReport
- minerId
- resourcesAssigned
- resourcesMined
- resourcesMissing
- bonuses
- penalties
- finalSalary
```
